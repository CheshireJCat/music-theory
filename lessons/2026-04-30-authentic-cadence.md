# 2026-04-30：正格终止 Authentic Cadence

## 今日知识点

昨天你已经知道 `V7` 为什么会强烈地想回到 `I`。今天把这个现象再整理成一个完整概念：当 `V` 或 `V7` 走向 `I`，并让乐句听起来真正“结束”时，这就叫 **正格终止**（Authentic Cadence）。

以 `C` 大调为例，最典型的正格终止是：

```text
G -> C
或
G7 -> C
```

重点不只是“前一个和弦接后一个和弦”，而是听感上的方向非常明确：

- `V` 或 `V7` 先制造紧张感
- `I` 提供稳定落点
- 如果是 `V7 -> I`，终止感通常会更清楚

你可以先把它理解为：**正格终止就是最常见的“回家式结尾”**。很多儿歌、流行歌、副歌尾句、钢琴初级伴奏，最后之所以像真的结束了，往往就是因为出现了这种终止。

![钢琴上的正格终止示意](../assets/2026-04-30-authentic-cadence/piano-authentic-cadence.png)

如果用昨天的知识继续往下看，会发现正格终止其实就是把 `V7 -> I` 的解决感，提升成“乐句结构”层面的理解。也就是说，昨天你学的是“为什么会想解决”，今天学的是“这种解决为什么会构成结尾”。

![正格终止的结构流程图](../assets/2026-04-30-authentic-cadence/authentic-cadence-flow.png)

## 钢琴使用场景

钢琴上学正格终止非常直接，因为你可以同时看到和听到“张力 -> 解决”的全过程。

- 左手弹低音 `G -> C`
- 右手弹 `G-B-D-F -> C-E-G`
- 弹完以后故意停顿半拍，体会 `C` 是否有“落稳”的感觉

![钢琴上的正格终止示意](../assets/2026-04-30-authentic-cadence/piano-authentic-cadence.png)

实际使用里，钢琴常见场景有：

- 一句旋律结束时，用 `G7 -> C` 收尾
- 左手分解伴奏弹完整句子，最后两拍改成 `V7 -> I`
- 给旋律配和声时，想让最后一小节更像结尾，而不是“还要继续”

如果你在钢琴上弹一串和弦，比如 `C - F - G7 - C`，最后回到 `C` 时的稳定感，通常会明显强于停在 `G7` 上。这就是正格终止最核心的实际价值。

## 吉他使用场景

吉他里，正格终止常常不是用来“展示一个理论术语”，而是直接出现在弹唱结尾。

- 在 `C` 大调弹唱中，最后一小节常用 `G7 -> C`
- 副歌唱完后，编配常把最后一个属和弦拉满，再回主和弦
- 民谣扫弦里，如果最后只停在 `G`，句子可能还像没说完；回到 `C` 才像句号

![吉他的正格终止示意](../assets/2026-04-30-authentic-cadence/guitar-authentic-cadence.png)

对吉他来说，最实用的训练不是背定义，而是对比：

- 只弹 `| C | F | G7 |`
- 再弹 `| C | F | G7 | C |`

第二种会明显更像“完整结束”，这就是正格终止的听觉特征。

## 可演奏例子

钢琴版本：

```text
例子 1：最基本的终止
左手：G        C
右手：G-B-D-F  C-E-G

例子 2：放进四小节
左手：C    F    G    C
右手：C和弦 F和弦 G7和弦 C和弦
最后两拍特别听 G7 -> C
```

吉他版本：

```text
例子 1：基础终止
| C | F | G7 | C |

例子 2：对比句尾
先弹：| C | F | G7 |
再弹：| C | F | G7 | C |
比较哪一个更像“句子结束”
```

## 今日练习

1. 在钢琴上连续弹 8 次 `G7 -> C`，每次回到 `C` 后停一拍，专门听“落稳”的感觉。
2. 用钢琴弹 `C - F - G7 - C`，尝试判断哪一个和弦最像句尾前的“准备动作”。
3. 在吉他上循环 `| C | F | G7 | C |` 4 轮，保持稳定拍子，不要急着换和弦。
4. 在吉他上对比 `| C | F | G | C |` 和 `| C | F | G7 | C |`，听哪个终止感更强。
5. 自己哼一条 4 小节旋律，最后配成 `G7 -> C`，确认它是否比停在 `G7` 更像真正结尾。

## 一句话总结

正格终止就是 `V` 或 `V7` 回到 `I` 所形成的结尾感，它把“属功能的张力”明确地解决到“主和弦的稳定”。
