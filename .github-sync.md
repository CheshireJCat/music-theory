# GitHub Sync Notes

Remote repository: `https://github.com/CheshireJCat/music-theory`

Daily automation should call:

```bash
/Users/nekos/daily/music-theory/scripts/sync_to_github.sh
```

The script stages the generated lesson archive content and GitHub Pages entry files, commits them when there are changes, rebases on `origin/main` when that branch exists, and pushes to GitHub.
